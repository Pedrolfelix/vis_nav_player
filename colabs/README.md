# 📓 Course Labs — ROB-GY 6203: Robot Vision

> Weekly hands-on implementations submitted as coursework for **Robot Vision** at NYU Tandon School of Engineering.

Each notebook/folder corresponds to a hands-on session covering a core computer vision topic, implemented in Python using Google Colab.

---

## 🗂️ Labs Overview

| # | Topic | Key Concepts |
|---|-------|--------------|
| 01 | Intro to Robot Vision | Course setup, image basics |
| 02 | Intro to Deep Learning | Neural networks, PyTorch fundamentals |
| 03 | Camera Calibration | Intrinsic/extrinsic parameters, chessboard calibration, lens distortion |
| 04 | Tag-Based AR | AprilTag detection, homography, pose estimation |
| 05 | Stereo Vision | Disparity maps, depth estimation, epipolar geometry |
| 06 | Structure from Motion (SfM) | Feature matching, 3D reconstruction from 2D images |
| 07 | ICP | Iterative Closest Point, point cloud alignment |
| 08 | RANSAC | Robust estimation, homography under outliers |
| 09 | Object Detection | Detection pipelines, bounding box prediction |
| 10 | Tracking | Multi-object tracking algorithms |
| 11 | Segmentation | Semantic and instance segmentation |
